## 二.寻找h元素和p元素的案例

核心结论：h 元素用于定义标题（h1-h6 层级递减），p 元素用于定义段落文本，二者是 HTML 基础内容元素。

#### 实际案例代码

```html
<!DOCTYPE html>
<html>
<head>
    <title>h与p元素示例</title>
</head>
<body>
    <h1>网页主标题（h1，最高层级）</h1>
    <p>这是围绕主标题展开的第一段正文内容，p元素会自动在前后添加空白，实现段落分隔。</p>
    
    <h2>二级标题（h2，次级层级）</h2>
    <p>这是二级标题对应的段落文本，用于详细说明该小节的核心信息，提升内容可读性。</p>
    
    <h3>三级标题（h3，更细分层级）</h3>
    <p>p元素支持换行显示长文本，无需额外标签，浏览器会自动处理文本换行和间距。</p>
</body>
</html>
```

## 三.寻找a和img(3个)

核心结论：a 元素用于创建超链接，img 元素用于插入图片，二者是实现网页交互和可视化的关键元素。

#### 3 个实际案例代码

```html
<!DOCTYPE html>
<html>
<head>
    <title>a与img元素示例</title>
</head>
<body>
    <!-- 示例1：a元素链接到外部网页，img元素插入本地图片 -->
    <a href="https://www.baidu.com" target="_blank">访问百度（新窗口打开）</a>
    <img src="local-image.jpg" alt="本地风景图" width="300" height="200">
    
    <!-- 示例2：a元素链接到网页内部锚点，img元素插入网络图片 -->
    <a href="#section1">跳转到页面第一部分</a>
    <img src="https://picsum.photos/400/300" alt="随机网络图片" title="点击查看大图">
    
    <!-- 示例3：a元素包裹img，实现图片链接，img元素添加加载失败提示 -->
    <a href="https://www.github.com" target="_self">
        <img src="github-logo.png" alt="GitHub图标（加载失败时显示）" width="100" height="100">
    </a>
    <div id="section1">页面第一部分内容（锚点目标位置）</div>
</body>
</html>
```

## 四.说出div和span元素的作用和区别

核心结论：div 和 span 均为容器元素，无默认样式，核心区别在于显示方式和使用场景。

#### 1. 核心作用

- div：用于**块级布局**，划分网页大的结构区域（如头部、主体、底部），承载多个相关元素。
- span：用于**行内修饰**，包裹文本或行内元素（如关键词、图标），实现局部样式调整。

#### 2. 关键区别

| 特性     | div 元素                 | span 元素                  |
| -------- | ------------------------ | -------------------------- |
| 显示方式 | 块级元素，独占一行       | 行内元素，与其他元素同行   |
| 适用场景 | 页面结构划分、大区域布局 | 文本局部修饰、行内元素分组 |
| 默认样式 | 无，但自动换行           | 无，不自动换行             |
| 嵌套规则 | 可嵌套块级、行内元素     | 仅可嵌套行内元素或文本     |

## 五.HTML全局属性有哪些?分别是什么作用?

核心结论：HTML 全局属性可用于所有 HTML 元素，核心作用是统一控制元素的通用行为或样式。

#### 常用全局属性及作用

1. id：给元素分配唯一标识，用于 JS 获取元素或 CSS 精准选择，同一页面不可重复。
2. class：为元素定义一个或多个类名，用于 CSS 批量样式控制，多个类名用空格分隔。
3. style：直接在元素上定义行内 CSS 样式，优先级高于外部 CSS，格式为 “属性：值”。
4. title：鼠标悬浮在元素上时显示的提示文本，常用于补充元素说明。
5. hidden：设置元素是否隐藏，值为 “hidden” 时元素不显示且不占页面空间。
6. lang：指定元素内容的语言（如 “zh-CN” 表示中文），帮助浏览器和搜索引擎识别。
7. data-*：自定义数据属性，用于存储页面私有数据（如 data-id="123"），可通过 JS 读取。
8. tabindex：控制元素是否可通过 Tab 键聚焦，数值越大聚焦优先级越低（负数表示不可聚焦）。
